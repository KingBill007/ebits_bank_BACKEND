# ebits_bank_BACKEND
**how to run**
- Download node js and install
- open directory in terminal and run command " npm install "
- after install run command " npm start "
- server will start

you need to install the database aswell
- download MongoDB Community Server, you can get it from https://www.mongodb.com/try/download/community
- install using the msi package
- during installation check the download mongoDB compass option to download the GUI for the db
- you can get it here aswell https://www.mongodb.com/products/tools/compass
- make sure its done before you run the server. after the server is run open compass to view your database
